BangLang

BangLang is a Bengali Python Interpreter created by Suryalal Singha. Using BangLang, you can write Python programs using Bengali keywords!

About the Project

This is a simple and experimental project aimed at making Python programming accessible in Bengali. It translates Bengali keywords into Python keywords and executes the code. It is a personal dream project of the developer.
